# Фронтенд проекта Speech Performance Analyzer

*Веб-интерфейс для анализа устных выступлений с визуализацией эмоций и генерацией отчёта*

**Технологии и инструменты**

- React 19 с TypeScript
- Vite как сборщик и dev-сервер
- Tailwind CSS для стилизации
- Recharts для графика эмоций
- Локальное хранение истории через localStorage

**Основные компоненты**

* Sidebar — боковая панель с историей анализов, поиском и кнопкой нового анализа
* FileUpload — начальная страница с drag & drop загрузкой файла и описанием возможностей
* MediaPlayer — проигрыватель аудио/видео с возможностью выбора фрагмента через кастомный RangeSlider
* AnalysisResults — страница с результатами: метрики, отчёт от LLM, график эмоций, транскрибация
* EmotionChart — график на основе Recharts, отображающий динамику эмоций во времени
* Набор SVG-иконок (React-компоненты): UploadIcon, SaveIcon, HistoryIcon, TrashIcon, PlusIcon

**Особенности**

* Полная адаптивность под мобильные и десктопные устройства
* Интерфейс переведён на русский язык
* Все состояния управляются через useState и useEffect в App.tsx
* API-вызовы к бэкенду выполняются через services/apiService.ts
* Типизация строгая, через файл types/index.ts

**Запуск в режиме разработки**

1. В папке frontend выполнить
npm install
2. Затем
npm run dev
3. Открыть в браузере http://localhost:3000
