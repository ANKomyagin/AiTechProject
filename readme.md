# Speech Performance Analyzer
— это приложение для комплексного анализа устных выступлений. Оно предназначено для ораторов, преподавателей, тренеров и студентов, которые хотят объективно оценить эмоциональную выразительность, структуру и качество речи.

**Проект может использоваться как**:

- Инструмент саморефлексии для выступающих
- Образовательный тренажёр по ораторскому искусству

**Проект реализует полный pipeline анализа**:

* Распознавание речи и разделение по спикерам (диаризация) на основе WhisperX и pyannote.audio.
* Анализ эмоциональной окраски каждого фрагмента выступления через WavLM и обученную MLP-модель.
* Генерация экспертного текстового отчёта с помощью локальной языковой модели qwen2.5:3b, запущенной через Ollama.
* Визуализация результатов в веб-интерфейсе: график динамики эмоций, транскрибация с тайм-кодами, метрики качества.
* Приватность: все вычисления происходят на устройстве пользователя. Аудиофайлы не покидают локальную машину — даже временные файлы удаляются сразу после обработки.

Система модульна: бэкенд (FastAPI) и фронтенд (React) независимы, что позволяет легко заменять компоненты — например, подключить другую LLM или использовать заглушки для демонстрации.
