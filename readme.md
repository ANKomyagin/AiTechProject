# Настройка виртуальной среды 

Необходимые зависимости рекомендуется установить через requirements.txt. Они не являются оптимальными,
но стабильно работают.

Для корректной работы whisperx необходимо, чтобы был установлен ffmpeg. 
Для работы whisperx на видеокарте также требуется установить cuda.
